Waar laat je edge liggen? 👻

Missed trades zijn setups die je spotte maar niet nam. Door ze los te loggen met een reden krijg je antwoord op: ben ik te conservatief? · is m'n edge-selectie juist? · mis ik consistent op bepaalde tijden of setups? Onmisbaar voor wie serieus aan proces werkt.

🔕 Opt-inMaster toggle in Instellingen. Default UIT — nooit in de weg voor users die het niet willen.
🧩 Apart gelayerdVier lagen zichtbaarheid: feature uit / aan maar verborgen / filter aan / analytics aan. Elke user kiest z'n diepte.
📊 Nooit in echte P&LMissed trades tellen nooit mee in je win-rate of netto PnL. Aparte analytics sectie "Edge Gap".
💾 Data-veiligToggle uit = verborgen, niet gewist. Backup-export bevat ze automatisch. Zet 'm weer aan → alles terug.
1. Master toggle in Instellingen

Zonder deze toggle zie je van missed-trades niks. Mét deze toggle verschijnen UI-elementen op de juiste plekken.

👻 Gemiste trades bijhouden UIT
Default voor nieuwe users. Alsof de feature niet bestaat:
  • Geen 👻 toggle in TradeForm
  • Geen filter-pill in Trades-lijst
  • Geen "Edge Gap" sectie in Analytics
  • Geen "Missed-redenen" categorie in TagManager
  • Geen ⌘K → m command
Zero cognitive load voor wie alleen echte trades wil loggen.
👻 Gemiste trades bijhouden AAN
Alle UI-elementen verschijnen, maar:
  • Trade-filter default op "Genomen" (missed blijft verborgen tot je wil zien)
  • Edge Gap sectie kan apart verborgen via layout-gear ⚙
  • Reden-tags zijn aangeraden, niet verplicht
Power-feature voor wie serieus aan edge-tracking wil.

5. TagManager in Instellingen → Tags

Missed-redenen leven in dezelfde tag-infrastructuur als setupTags / emotionTags. Volledig bewerkbaar — user kan eigen redenen toevoegen.

Deze categorie verschijnt alleen als 👻 Gemiste trades bijhouden aan staat in Instellingen → Accounts. Je kunt eigen redenen toevoegen (bv. "losstreak-pauze", "wachten op 4H-confirmatie").

Edge Gap analytics

Nieuwe sectie: captured-ratio tabel, reasons donut, edge-leak stat. Setup Insights extra kolom.

⏱ ~3 uur
FASE 4

Hindsight + polish

Optional hindsight-exit, auto R-multiple. Heatmap "Gemist alleen" filter. Weekly-review nudge.

⏱ ~2 uur

Totaal ~10 uur verspreid over 4 releases. Elke fase is op zichzelf bruikbaar; community feedback per fase meenemen.
