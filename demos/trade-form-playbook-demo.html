💡 Wat deze demo laat zien
1. Playbook-selectie auto-vult setup-tags / timeframes / sessies / confirmaties — gevlamde groene auto-active styling laat zien wat er net is gevuld (animatie). User mag alles handmatig overrulen — pills blijven toggelbaar.

2. Criteria-checklist verschijnt onderaan zodra een playbook is gekozen. Elk criterium heeft een verplicht of optioneel label.

3. Compliance-score rekent live op basis van mandatory criteria afgevinkt: ≥80% = groen, 50-79% = amber, <50% = rood. Save-knop kleurt mee — geen blokkade, alleen visueel feedback.

4. Schema-impact: trade krijgt 3 nieuwe velden — playbookId, complianceChecks[], complianceScore. Backwards-compat voor bestaande trades (default empty).

5. Compliance × PnL split wordt harde data — geen heuristiek meer ("matcht confirmation-tag?") maar exact: "trade had 71% mandatory criteria afgevinkt → +0.4R real outcome".
